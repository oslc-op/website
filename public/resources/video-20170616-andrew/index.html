New and updated tools in Eclipse Lyo: Toolchain Designer & Lyo Store

Jun 16, 2017

No comments

In this video, Andrew Berezovskyi, a Lyo project lead will describe the latest support tools that were contributed to Eclipse Lyo, which will further aim the development of OSLC adaptors and toolchains.

Lyo is an Eclipse project aimed at helping the community adopt OSLC specifications and build OSLC-compliant tools. Learn more about Lyo on https://www.eclipse.org/lyo/
